Projects & Products

btcfile.com

btcfile.com is a registration-free downloadable content marketplace. Visitors can instantly sell or buy any kind of digital content for bitcoins.

deviantART Submission

Under contract for deviantART, we worked on a complete overhaul of the submission experience. Going from an old html form to a full-featured HTML5 experience.

deviantART World Map

Under contract for deviantART, we created a real-time world map of all user activity on deviantart.com and sta.sh.

deviantART APIs

Under contract for deviantART, we developer the first set of APIs to let developers develop applications leveraging deviantart.com and sta.sh.

sta.sh

Under contract for deviantART, we worked on sta.sh, a standalone HTML5 website providing asset upload and instant sharing services.

inspi.re

Inspi.re, which ran from 2008 to 2013, was a photo competition social network. Users created communities, ran competitions and voted on each other's contributions daily. Its source code is available on github.
